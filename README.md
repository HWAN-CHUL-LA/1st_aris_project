# RAG를 활용한 AI 챗봇 기반 아이스크림 제조 로봇 서비스  
---
ai 가 rag, stt 에 따라 

## 주요 기술 구성요소 
1. Large Language Model (LLM) + Retrieval-Augmented Generation (RAG)
2. FastAPI 서버 (UI 화면-백엔드 연동)
3. DB: 고객 얼굴 및 데이터 관리
4. Face Recognition
5. Object Detection
6. Robot Motion
